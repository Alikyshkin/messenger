# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Добавлено
- Автоматическая проверка кода перед коммитом (pre-commit hooks)
- CI/CD проверки безопасности зависимостей
- Dependabot для автоматических обновлений зависимостей
- Шаблоны для Pull Requests и Issues
- Проверка размера бандла в CI

### Изменено
- Унифицированы UI компоненты и стили
- Улучшена навигация с боковой панелью

## [1.0.0] - 2026-02-11

### Добавлено
- Регистрация и вход пользователей
- Чаты и переписка в реальном времени
- Голосовые и видеозвонки (WebRTC)
- Контакты и заявки в друзья
- Шифрование сообщений (E2EE)
- Офлайн режим с синхронизацией
- Поддержка вложений (файлы, изображения, аудио)
- Реакции на сообщения
- Групповые чаты
- Медиа-галерея
- Уведомления об обновлениях приложения
